# Público alvo

## Nome:
Alvenir Ribeiro Luz

## Localização:
Rua tainar pires 27 - Cajazeiras

## Contexto e necessidades:
Dona alvenir é dona de uma pequena lanchonete localizada no bairro Cajazeirase vem enfrentando dificuldades em organizar suas finanças de forma efíciente, necessitando de uma ferramenta que facilite isso.
